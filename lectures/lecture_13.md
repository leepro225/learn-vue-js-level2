# 프로젝트 생성 도구 - Vue CLI

### 공식 사이트 링크
https://cli.vuejs.org/

      node -v  // node 버전 확인 10점대
      npm -v   // npm 버전 확인 6점대
      npm install -g @vue/cli   //CLI 설치
      sudo npm install -g @vue/cli   // permision error가 뜨는 경우


### npm이 package install 하는 곳
/nodemodules


### 프로젝트 생성하기

      vue create something //뷰로 만들 폴더 명
      
 >default 선택
 
      cd something //이동해서
      npm run serve //실행하면됑
